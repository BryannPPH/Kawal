# Garuda Site Ops

React + TypeScript + Vite dashboard prototype for IoT site telemetry, risk scoring, intervention planning, inspection tasks, and local activity logging.

## Stack

- React + TypeScript + Vite
- Tailwind CSS
- Zustand with `localStorage` persistence
- Lucide React icons
- Vitest
- SVG map with absolute-positioned site zones

## Scripts

```bash
bun install
bun run dev
bun run build
bun run test
```
